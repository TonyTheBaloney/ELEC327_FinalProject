NeuralSeed

Neural amp simulation
on bare metal

A GRU10 model captured from real hardware — Klon, TS9, Marshall, and more — runs entirely on the Cortex-M7 using RTNeural, a zero-overhead header-only inference engine. Model weights are compiled directly into the firmware binary.

Getting Started

Build & flash

Uses a reproducible Nix dev shell — no manual toolchain installation required.

01
Clone the repository

git clone https://github.com/TonyTheBaloney/ELEC327_FinalProject

02
Pull submodules

git submodule update --init --recursive — fetches libDaisy, DaisySP, and RTNeural.

03
Enter the Nix dev shell

From AudioProcessing/: nix develop --extra-experimental-features "nix-command flakes"

04
Build libraries & firmware

make -C libDaisy && make -C DaisySP && make

05
Flash to Daisy Seed

Put the board in DFU mode, then make program-dfu.
